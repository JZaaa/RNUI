swipeout API

注意：同时仅能打开一个滑动列表

拓展方法 说明
$(selector).swipeout('open') 打开滑动列表
$(selector).swipeout('close') 关闭滑动列表

使用示例

$('.weui-cell_swiped').swipeout()
$('.weui-cell_swiped').swipeout('open')
$('.weui-cell_swiped').swipeout('close')
