weui.js custom

修改weui.js 源码，添加多种提醒样式

代码演示

weui.toast('操作成功', 3000);
weui.toast('操作成功', {
    duration: 3000,
    className: 'custom-classname',
    style: 'ok',
    callback: function(){ console.log('close') }
});
weui.toast('取消操作', 'cancel');
weui.toast('操作成功', 'ok');
weui.toast('操作失败', 'warning');

toast API

参数 说明 类型 默认值
content toast的文字 string -
options 配置项或回调 Object | function -
[options.duration] 多少毫秒后关闭toast number 3000
[options.callback] 关闭后的回调 function -
[options.className] 自定义类名 string -
[options.style] 提醒类型,cancel | warning | ok | null string null
