js

weui.slider('#slider', {
    step: 10,
    defaultValue: 40,
    onChange: function(percent){
        console.log(percent);
    }
});
参数 说明 类型 默认值
selector 选择器 string -
options 配置项 Object -
[options.step] slider的step，每次移动的百分比，取值范围 [0-100] number -
[options.defaultValue] slider的默认百分比值，取值范围 [0-100] number 0
[options.onChange] slider发生改变时返回对应的百分比，取值范围 [0-100] function -
