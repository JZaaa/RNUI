jzTabbar API

参数 说明 类型 默认值 可选值
actionIndex 当前选中标签的索引 false | number false -
actionClass 活动状态class string tabbar-item-active -
item 子标签选择器 string .tabbar-item -

使用示例

    $(selector).jzTabbar({
        activeIndex: false,
        activeClass: 'tabbar-item-active',
        item: '.tabbar-item'
    })
