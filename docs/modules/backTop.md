返回顶部 backTop
--------------------
### 使用方法

````javascript
new rn.BackTop(selector, options)
$(selector).backTop(options)
````

### 参数

| 参数      |    介绍 | 类型  | 可选值 | 默认值 |
| :--: | :--:| :--:|:--:   | :--: |
| showHeight  | 当滑动此位置时显示目标 |  number   | - | 400 |
| backPosition  | 返回顶部位置 |  number   | - | 0 |
| display  | 强制目标显示display样式 |  string   | - | - |
